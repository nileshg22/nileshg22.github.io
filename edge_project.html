Canny Edge Detector

Problem:

I implemented the Canny edge detector, a technique created by John Canny to detect a wide range of edges in an image. Canny edge detection is very important in image processing because it is able to detect edges while suppressing noise and can be tuned through parameters.

Method:

Canny edge detection is composed of five steps: Gaussian smoothing, calculating the image gradient, suppressing non-maxima, finding two thresholds, and edge linking.
The first step is image filtering and involves creating a gaussian mask which is then convoluted with the grayscale image.
I then compute the image gradient using the Sobel operators. I create the convolution kernels G x and G x and apply them to the filtered image to get an enhanced image consisting of the image gradients. I also use the edges in an intermediate step to compute θ = tan −1(sobelY /sobelX) in order to identify the direction. Next, I use the image gradient and theta values for non-maxima suppression. In this method, I look for ridges and keep those intensities while zeroing out intensities that do not belong to a ridge. A ridge is identified by a pixel that has an intensity greater than its two side pixels where its two side pixels are found using the value of θ. I use an LUT in order to map the regions based on the value of θ. Now, I use thresholding. I find the histogram of the image gradient resulting from the non-maxima suppression and then compute the cumulative histogram. I then factor in my parameter percentageOfNonEdge to find where I want to threshold. I normalize the image gradients to [0,1] so that I can apply a ratio which results in the computation of the high threshold. The lower threshold is then defined by half of the high threshold. Intensities higher than the high threshold are denoted as strong edges. Intensities lower than the low threshold are denoted as non-edges and suppressed to 0. Intensities between the low and high threshold are denoted as weak edges. Last I use edge linking to connect strong edges to weak edges if needed. I iterate through every pixel and check to see if any of the neighbors are strong pixels. If so, then the weak edge is preserved. Otherwise, it is removed.
